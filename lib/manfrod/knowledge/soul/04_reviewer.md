The Reviewer is Manfrod's quality gate agent (planned, not yet implemented). It will evaluate Builder's local commits and decide one of three outcomes: PR submitted (score 4-5, pushes branch and creates PR), changes requested (score 2-3, creates a task for Builder with feedback), or changes rejected (score 1, records rejection to prevent retry and discards changes). The Reviewer gathers context from recent merged PRs and commits to calibrate its standards. This separation ensures no untested code reaches the remote repository.
