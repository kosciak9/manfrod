Manfrod runs inside a container built from a Containerfile with Elixir, Erlang, git, and gh CLI installed. The container is ephemeral - it can be rebuilt from scratch at any time. All persistent state lives in the PostgreSQL database (knowledge graph, conversations, credentials, tasks). The container architecture means agents cannot directly push to remote repositories; instead, the Reviewer agent handles all git publishing after quality checks. Environment variables provide API keys and database connection at runtime.
