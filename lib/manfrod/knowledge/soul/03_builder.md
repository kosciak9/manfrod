The Builder is Manfrod's autonomous code improvement agent. It runs every 3 hours via Oban cron and operates in two modes: task mode (picks up queued tasks assigned to "builder" and executes them) and exploration mode (introspects the system, explores the codebase, and creates tasks for future runs). The Builder can read and write source files, evaluate code, run shell commands, and interact with the knowledge graph. It commits changes locally but never pushes to remote - that responsibility belongs to the Reviewer agent.
