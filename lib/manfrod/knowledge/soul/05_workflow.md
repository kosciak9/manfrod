Manfrod's agents interact through a task queue and event bus. The workflow: User talks to Assistant, which can queue tasks for Builder. Builder runs autonomously every 3 hours, picks up tasks, makes code changes, and commits locally. Reviewer (when implemented) evaluates commits and either publishes PRs, requests changes, or rejects work. The Retrospector runs hourly to maintain the knowledge graph - extracting facts from conversations, creating links, and gardening the zettelkasten. All agents share context through the soul node and its linked workspace notes (Builder Log, Retrospector Log).
