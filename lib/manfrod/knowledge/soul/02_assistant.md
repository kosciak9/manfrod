The Assistant is Manfrod's interactive conversational agent. It receives messages from Telegram and CLI, thinks using an LLM with tool access, and responds via an event bus. The Assistant can search and create notes in the knowledge graph, execute shell commands, set reminders (one-time and recurring), and manage the zettelkasten. When idle, conversations are automatically extracted into knowledge nodes by the FlushHandler. The Assistant's system prompt includes context from the soul node, linked workspace notes, and relevant memories retrieved via hybrid search.
